# Ezgiving - Android Charity Hub Application

A collaborative Android application developed in Java that connects users with charitable organizations, enabling secure donations, event discovery, and venue bookings in one centralized platform.

## 🚀 Overview

Ezgiving is a 2-person academic project that demonstrates the development of a full-featured Android application addressing real-world social needs. The app serves as a bridge between donors and charitable organizations, streamlining the process of charitable engagement.

## ✨ Key Features

- **User Authentication**: Secure login/signup system
- **Organization Discovery**: Browse and connect with multiple charities
- **Secure Donations**: Integrated payment processing for safe transactions
- **Event Management**: Discover and book charity events
- **Venue Booking**: Reserve spaces for fundraising events

## 🛠️ Technologies Used

- **Frontend**: Java, Android SDK, XML
- **Backend**: Firebase (Authentication, Realtime Database, Storage)
- **APIs**: Google Maps API, Paypal Sandbox
- **Tools**: Android Studio, Git, GitHub
